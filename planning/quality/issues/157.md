**Status:** quality-approved (second pass, 2026-04-09)

## Summary
Add CaptureHealthSnapshot to HUDManager: mic permission, input device, transcription provider, static LatencyTier.

## Plan core (unchanged from 2026-03-25)
1. `CaptureHealthSnapshot` as distinct `@Published var captureHealth` on `HUDManager`, in `SpeakApp` only.
2. Plain struct, value semantics — no ObservableObject, no source manager references.

4. Three update triggers only: `PermissionsManager.statuses`, `AudioInputDeviceManager`, `AppSettings.liveTranscriptionModel` — via dedicated `updateCaptureHealth(_:)` method.
