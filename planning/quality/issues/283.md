# Issue #283 — fix(ios): add missing SpeakCore import

## Status: approved (first pass)

Root cause: `SpeakiOSApp.swift` imports only `SpeakiOSLib`; `OpenClawClient` is defined in `SpeakCore`.
Fix: add `import SpeakCore` at line 2 of `SpeakiOSApp/SpeakiOSApp.swift`.
